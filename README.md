# Framingham-heart-study-analysis
Data analysis of Framingham heart study using python
## Overview
This project analyzes the Framingham Heart Study dataset to identify
cardiovascular disease risk factors using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Files in this Repository
- EDA_HeartDisease.ipynb : Python notebook with data analysis
- DiseasePredictionReport-MahimaShivraj : Project report summarizing methods and results

## Key Work
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Statistical analysis of risk factors
